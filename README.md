# Site Base — Griffin Phillips & Voe Campos

Template estático, leve e responsivo (HTML/CSS/JS puro) — pronto para publicar no GitHub Pages, Netlify ou Vercel.

## Como usar
1. Envie a pasta para um repositório.
2. Publique no GitHub Pages (branch `main`) ou arraste a pasta no Netlify.
3. Edite os números de WhatsApp nos arquivos `.html` (procure por `wa.me/55`).

## Personalização rápida
- **Cores**: edite variáveis em `assets/css/style.css` (`--bg`, `--brand`, etc.).
- **Logo**: troque `favicon.svg` e adicione `logo.svg` se quiser.
- **Imagens**: substitua `assets/img/hero.jpg` e adicione outras.
- **SEO**: ajuste `title`, `description` e `og:*` no `<head>`.
